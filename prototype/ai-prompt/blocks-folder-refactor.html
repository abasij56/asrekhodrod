گزارش پیاده‌سازی — تم عصر خودرو — ۲۹ بلوک با auto-discovery از طریق BlockRegistry

تغییرات سیم‌کشی کلیدی

  1. BlockRegistry — اسکن inc/blocks/*/config.php، ساخت manifest entries، بوت ACF blocks
  2. load.php — بوت واحد؛ بدون لیست require_once برای هر بلوک
  3. Appearance.php — مسیرهای Timber شامل inc/blocks؛ resolve_template() پوشه بلوک را هم چک می‌کند
  4. Appearance manifests — کلید blocks اکنون array() (فقط override در صورت نیاز)
  5. LayoutSchema::shared_blocks() — به BlockRegistry::manifest_entries() واگذار شد
  6. LayoutResolver — partialها از طریق BlockRegistry::resolve_partial()
  7. Legacy Blocks::init() (ak-* ACF homepage blocks) — حذف شد
  8. Carsinfo seed order — با seed_order در هر cinfo config.php حفظ شد

تغییرات شکننده (Breaking changes)

تغییرتأثیر
حذف legacy acf/Blocks.php ak-* ACF registration بلوک‌های Gutenberg قدیمی ak-* در محتوای پست (در صورت وجود) رندر نمی‌شوند؛ بلوک‌های layout builder تحت تأثیر نیستند
مسیر Timber inc/blocks/acf → inc/blocks ارجاعات hard-coded به مسیر قدیمی می‌شکنند (در تم یافت نشد)
حذف manifest-blocks.php کد خارجی که آن فایل را require می‌کند می‌شکند (فقط داخلی تم بود)

هیچ تغییر رفتاری عمدی در front-end برای layout zones، page keys، یا cinfo blocks وجود ندارد.

چک‌لیست تست دستی

  • Front page (Classic) — ad strip, ticker, hero, featured, news list, magazines, videos, reviews, newsletter, sidebar ads, picture frame
  • Front page (Modern Blue) — hero/featured/videos/newsletter/picture-frame از overrideهای MB
  • Archive — sidebar widgets + ad strip
  • Single post — sidebar widgets + ad strip
  • Page about / contact / 404 — block page content رندر شود
  • Layout builder admin (چیدمان صفحات) — همه بلوک‌های قابل قرارگیری لیست شوند؛ ذخیره/بارگذاری zone placement
  • Carsinfo single — هر ۹ بلوک cinfo رندر؛ TOC؛ gallery lightbox
  • New carsinfo post — auto-seed به ترتیب: hero → facts → overview → video → table → gallery → faq → related → comments
  • Rfarda appearance — front page hero/videos/reviews

قرارداد پوشه هر بلوک (هدف)

inc/blocks/{block-name}/
  config.php       — name, label, source, partial/template paths, defaults, seed_carsinfo (if ACF), type: layout|acf
  Block.php        — registration (ACF blocks only)
  Fields.php       — ACF field groups (acf blocks only)
  View.php         — Twig context builder (when needed)
  template.twig    — default render template
  partial.twig     — layout partial (layout blocks)
