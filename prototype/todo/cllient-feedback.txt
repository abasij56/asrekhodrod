صفحه اصلی:
done 1-  سایز لوگو کمی بزرگتر باشد
done 2-  نوار جست و جو : گزینه نمایش همه نتایج، یا گزینه نمایش نتایج در صفحه دیگر ندارد
done 3-  تاریخ روز شامل روز، ماه و "سال" باشد
Done 4-  از منو نسخه دسکتاپ، تماس باما و درباره ما فقط در فوتر باشد، نیازی نیست بالای سایت هم باشد
** 5-  نیاز داریم سایت نسخه انگلیسی هم داشته باشد(با گوگل ترنسلیت یا هوش مصنوعی)
done 6-  نوار قرمز خبر فوری : اخبار از چپ به راست باید حرکت کنند الآن برعکس هست
done 7-  اسلایدر 5 خبر اصلی : رنگ زمینه روتیتر مطالب قرمز نباشد و مشابه رنگ کلمه "عصر" در لوگو عصر خودرو باشد 
Done 8-  بخش اخبار ویژه : این بخش نباشد بهتر است، در زمان برگزاری یک رویداد خاص یا نمایشگاه، امکان داشتن چنین بخشی را داشته باشیم اما اخبار به صورت نواری از راست به چپ نمایش داده شود، اون حالت ردن روی گوشه تصویر و باز شدن بخش جدید هم روی دسکتاپ هم  موبایل یکم عجیب هست و باگ داره
done 9-  بخش آخرین اخبار : در این بخش برای هر خبر صرفا روتیتر+تیتر+بخشی از مطلب کافی است و نیازی نیست گالری عکس ها هم زیر هر مطلب نمایش داده شود ( این مدل گالری که عکس تامبنیل دارن و روی عکس کلیک کنیم بزرگترش رو نشون میده رو داخل خود مطلب نیاز داریم نه در این قسمت)
Done 10-  در بخش آخرین اخبار صرفا نمایش 15 خبر آخر کافی است
Done 11-   قسمت مجلات روز : دیگر به این بخش در سایت نیازی نیست، ولی ممنون میشم بخش ویدیو ها مشابه بخش مجلات در زمینه مکی و با فرمکت مشابه نمایش داده بشن و اینکه آیا میشه تشخیص بده
Done 12-   قسمت در قاب تصویر حذف بشه
Done 13-  "https://e-rasaneh.ir/static/img/logo.jpg" در فوتر سایت فعلی عصر خودرو یک نماد E-Resane هست که باید در کد سایت جدید هم در فوتر نمایش داده بشه
** 14-   یک بخش نیاز داریم به اسم خودروها، اون صفحات لندینگ که برای هر خودرو یک صفحه می سازیم، می خوایم در این قسمت نمایش داده بشن در صفحه اصلی
Done 15-   نیاز به بنر تبلیغاتی استیکی در پایین صفحه داریم
done 16-  رنگ زمینه فوتر مشابه کلمه "عصر" در لوگو باشد
Done 17-  ما حتما نیاز به "آرشیو اخبار" با قابلیت انتخاب روز و ماه و سال و نمایش در قسمت منوی سایت داریم
Ask 18-  قسمت ارسال نظر رو درنسخه دسکتاپ ندیدم
Done 19-  لوگو سفید عصرخودرو برای فوتر ارسال می شه براتون
Done 20-  نمایش سوشال مدیاها در فوتر ( اگر دست خودمان است در پنل مشکلی نیست)

توضیح: اگر هر یک از بخش هایی که نیاز نداریم از طریق پنل خودمون قابل حذف و اضافه هست بفرمایید خودمون مدیریتش می کنیم، نمیدونم همه قابل کنترل هستن یا خیر
